## 2. 스토리

> 심해 자원 개발 기업이 운영하는 비밀 연구소에서 생체 실험을 당한 키메라가,  
> 더 깊은 심해로 도망치며 힘을 키워 연구소장에게 복수한다.

- 실험 과정에서 발생한 돌연변이 에너지로 인해 주변 심해 생물들도 변이되어 공격성을 띰
- 더 깊은 심해로 도망치며 다양한 생명체의 능력을 흡수해 성장
- 최종적으로 연구소로 돌아와 연구소장에게 복수

---

## 3. 스테이지 구성

### 스테이지 흐름
```
해저 연구소(탈출) → 해저 → 심해 → 해저 연구소(복수)
```

- 시작과 끝이 같은 장소 → 복수 서사의 완결성
- 처음엔 도망치던 연구소를 마지막엔 쳐들어가는 구조 → 캐릭터 성장 표현
- 1스테이지 연구소와 4스테이지 연구소는 시각적으로 다르게 표현 (불길, 파괴 등)

### 스테이지 상세

| 스테이지 | 배경 | 분위기 / 특징 | 보스 | 제한 시간 |
|---------|------|--------------|------|---------|
| 1 | 해저 연구소 | 탈출, 긴장감 / 좁은 맵, 장애물 | 실험용 실험체 | 8~9분 |
| 2 | 해저 | 광활, 미지 / 일반 필드 | 향유고래 | 8~9분 |
| 3 | 심해 | 어둠, 압박감 / 시야 제한 (원형 마스킹) | 산갈치 | 6~7분 |
| 4 | 파괴된 연구소 | 복수, 파괴 / 넓은 공간, 대량 적 | 연구소장 | 6~7분 |

> 각 스테이지 보스를 격파해야 다음 스테이지로 진행됨

## 4. 적 설계

### 1스테이지 — 해저 연구소

| 적 | 특징 |
|----|------|
| 방패 전투요원 | 느림, 체력 많음 |
| 근접 전투 보조요원 | 기본 |
| 원거리 제압요원 | 마취총 느낌, 투사체 느림, 체력 매우 낮음 |
| **보스: 실험용 실험체** | 느림, 가끔 일직선 돌진, 체력 25% 이하 시 폭주 (약간 빨라짐) |

### 2스테이지 — 해저

| 적 | 특징 |
|----|------|
| 피라냐 | 기본 |
| 청새치 & 상어 | 빠른 이동속도, 낮은 체력 |
| 전기뱀장어 | 약간 빠름, 범위 지속 데미지 |
| **보스: 향유고래** | 느림, 체력 많음, 가끔 초음파 공격 |

### 3스테이지 — 심해

| 적 | 특징 |
|----|------|
| 심해아귀 | 기본 |
| 대왕오징어 | 빠른 이동속도 |
| 심해해파리 | 약간 빠름, 자폭 (경고 모션 필요) |
| 풍선장어 | 발광 꼬리에서 원거리 공격 |
| **보스: 산갈치** | 매우 긺, 마디마디 발광효과, 돌진공격, 휘두르는 공격 |

### 4스테이지 — 파괴된 연구소 (강화된 전투요원들)

| 적 | 특징 |
|----|------|
| 강화 방패 전투요원 | 체력 많음, 느림, 돌진 |
| 원거리 전투요원 | 직선 레이저 공격 |
| 강화 근접 전투요원 | 빠른 이동속도, 강력한 공격 |
| **보스: 연구소장** | 인간+기계강화, 페이즈 변화 예정 |

> 연구소장 패턴: 드론 소환 (빠르고 자폭), 레이저 공격 (직선 원거리), 보호막

## 5. 독창적 시스템

### 5-1. 압력 시스템
- **적용 스테이지:** 해저(2), 심해(3)
- 하강할수록 압력 증가 → 이동속도 및 공격속도 감소 (페널티)
- 대신 경험치 획득량 증가 (리스크/리워드 구조)
- 메타 업그레이드(세포)로 최대 50%까지 저항 영구 강화 가능
- 답답함 방지: 압력 저항이 낮을 때는 어렵지만, 업그레이드하며 점점 극복하는 성장 서사 연출

### 5-2. 공격 수단 카피 시스템
- 스테이지 보스 격파 시 보스의 스킬 3개 중 1개 선택하여 카피
- 각 스킬은 공격형 / 생존형 / 특수형으로 분류
- 카피된 스킬은 슬롯에 장착: Q (1스테이지), E (2스테이지), 스페이스바 (3스테이지)
- 4스테이지 보스는 카피 대상 제외 (마지막 스테이지)

### 5-3. 생체 에너지 시스템
- 적 처치 시 생체 에너지 충전 (최대 200E)
- 카피한 보스 스킬 발동에 소모 (스킬마다 소모량 차등: 30E ~ 140E)
- 게이지 형태로 표시 → 전략적 운용 가능
- 에너지를 언제 쓸지 플레이어가 판단하는 구조

### 5-4. 돌연변이 진화 시스템
- 특정 레벨 구간에서 총 2회 돌연변이 선택 이벤트 발생
- 매번 2개의 선택지 중 1개 선택 / 중복 선택 불가
- 모든 선택지가 하이리스크·하이리턴 구조 (강력한 효과 + 단점 수반)
- 게임 정체성: "불안정한 키메라의 성장" → 안전한 선택지 없음

## 9. 메타 업그레이드 (세포)

- **재화명:** 세포
- **획득:** 게임 패배 또는 완료 후 성적에 따라 지급
- **사용처:**
  - 기본 스탯 영구 강화 (최대 HP, 이동속도, 공격속도, 공격력)
  - 압력 저항 강화 (최대 50%까지 단계별 업그레이드)
- 추후 시간 여유에 따라 스킬 해금 시스템 추가 고려

## 10. 캐릭터 비주얼

- **종류:** 키메라 (생체 실험체)
- **핵심 특징:** 사슬팔 (테오키스 + 알슈베르도 레퍼런스)
  - 뼈 마디마디가 사슬처럼 분리된 구조
  - 끝부분이 갈고리/날카로운 형태
  - 공격 시 사슬이 늘어나는 연출
- **색상:** 짙은 청록 + 흰 발광 포인트
- **피부:** 반투명, 생체 발광 점 패턴

### 돌연변이별 비주얼 변화

| 돌연변이 | 비주얼 변화 |
|---------|------------|
| 과부화 | 사슬이 굵어지고 붉게 발광 |
| 과성장 촉수 | 사슬이 여러 갈래로 분기 |
| 감각 붕괴 | 몸 전체 균열 + 파란 발광 |
| 의태 기관 | 몸이 반투명해지고 작아짐 |
| 독성 과부화 | 사슬에서 독액이 흘러내림 |

## 11. 개발 계획

### 기술 스택

| 항목 | 내용 |
|------|------|
| 엔진 | Unity 2022 LTS |
| 언어 | C# |
| 버전 관리 | Git / GitHub |
| 에셋 | Unity Asset Store 무료 에셋 + AI 이미지 생성 툴 활용 (Adobe Firefly, Bing Image Creator) |

### 구현 우선순위

| 단계 | 분류 | 구현 내용 | 예상 기간 |
|------|------|----------|---------|
| 1 | 핵심 기반 | 캐릭터 이동, 자동 공격, 적 기본 AI, 경험치/레벨업 | 1~2주 |
| 2 | 기본 시스템 | 스킬 시스템, 레벨업 선택 UI, 적 웨이브, 보스 구현 | 2~3주 |
| 3 | 독창 시스템 | 압력 시스템, 카피 시스템, 생체 에너지, 돌연변이 | 1~2주 |
| 4 | 스테이지/컨텐츠 | 4개 스테이지 맵, 심해 시야 제한, 적 밸런싱 | 1주 |
| 5 | 마무리 | 메타 업그레이드, UI/UX 정리, 버그 수정, 테스트 | 1주 |

> **핵심 원칙:** 에셋보다 코드 먼저. 흰 네모로 이동/자동공격 구현 후 에셋 붙이기 시작.

### 전체 일정 (약 6~8주)

| 기간 | 단계 | 주요 목표 |
|------|------|----------|
| 1~2주차 | 프로토타입 | 기본 이동, 자동 공격, 적 AI, 레벨업 / 플레이 가능한 최소 버전 완성 |
| 3~4주차 | 알파 버전 | 전체 스킬 구현, 독창적 시스템 4종 추가, 스테이지 1~2 완성 |
| 5~6주차 | 베타 버전 | 스테이지 3~4 완성, 보스 4종 구현, 메타 업그레이드 추가 |
| 7~8주차 | 완성/마무리 | 밸런싱, 버그 수정, UI/UX 개선, 최종 빌드 |

### 주요 기술 구현 방향

- **자동 공격:** Physics 2D OverlapCircle로 범위 내 적 탐색, 가장 가까운 적 자동 타겟팅
- **적 AI:** NavMesh 또는 벡터 추적 방식으로 플레이어 추적, ScriptableObject로 적 데이터 관리
- **스킬 시스템:** 각 스킬을 독립적인 컴포넌트로 모듈화, 레벨별 수치 ScriptableObject 분리
- **압력 시스템:** 플레이어 Y 좌표 기준으로 압력 수치 계산, 이동속도/공격속도에 곱연산 적용
- **심해 시야 제한 (3스테이지):** 2D Light 또는 Custom Shader로 캐릭터 주변 원형 마스킹
