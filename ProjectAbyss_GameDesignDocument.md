## 2. 스토리

> 심해 자원 개발 기업이 운영하는 비밀 연구소에서 생체 실험을 당한 실험체가,
> 더 깊은 심해로 도망치며 힘을 키워 연구소장에게 복수한다.

- 실험 과정에서 흘러나온 돌연변이 에너지로 인해 주변 심해 생물들도 변이되어 공격성을 띰
- 더 깊은 심해로 도망치며 다양한 생명체의 능력을 흡수해 성장
- 최종적으로 연구소로 돌아와 연구소장에게 복수

## 3. 스테이지 구성

### 스테이지 흐름
```
해저 연구소(탈출) → 해저 → 심해 → 해저 연구소(복수)
```

- 시작과 끝이 같은 장소 → 복수 서사의 완결성
- 처음엔 도망치던 연구소를 마지막엔 쳐들어가는 구조 → 캐릭터 성장 표현
- 1스테이지 연구소와 4스테이지 연구소는 시각적으로 다르게 표현 (탁한 녹회색 / 파괴)

### 스테이지 상세

| 스테이지 | 배경 | 분위기 / 특징 | 보스 | 보스 등장 / 제한 시간 |
|---------|------|--------------|------|---------|
| 1 | 해저 연구소 | 탈출, 긴장감 / 차가운 회청 톤 | 실험용 실험체 | 6분 / 7분 |
| 2 | 해저 | 광활, 미지 / 청록 톤, 압력 시작 | 향유고래 | 5분 / 6분 |
| 3 | 심해 | 어둠, 압박감 / 시야 제한(강한 비네트), 압력 | 산갈치 | 4분 / 5분 |
| 4 | 파괴된 연구소 | 복수, 파괴 / 탁한 녹회 톤, 대량 적 | 연구소장 | 4분 / 5분 |

> 각 스테이지 보스를 격파해야 다음 스테이지로 진행됨

### 동적 난이도 곡선 (앵커형 위기 연출)

- **정상 구간**: 시간 경과에 따라 스폰 밀도/적 체력이 선형으로 완만히 상승 (1·2스 기준 "딱 적당"하게 튜닝됨)
- **위기 구간**: 보스 등장 약 30초 전부터 스폰 밀도가 크레셴도로 치솟음 → 긴장 고조
- **BOSS INCOMING 경고**: 위기 구간 진입 시점에 HUD에 표시
- **보스 등장 시 완화**: 보스가 나타나면 일반 적 스폰이 완화되어 보스에 집중할 여유 제공
- **전역 강도 노브 / 스테이지별 조정**: 한 개의 배율 값으로 곡선 전체를 손쉽게 재밸런싱 (3·4스는 체력 스케일을 한 단계 낮춤)

## 4. 적 설계

### 1스테이지 — 해저 연구소 (연구소 직원 / SWAT)

| 적 | 특징 |
|----|------|
| 근접 전투요원 | 기본 근접 추적 |
| 방패 전투요원 | 느림, 체력 많음 |
| 원거리 제압요원 | 마취총 느낌, 투사체 느림, 체력 매우 낮음 (등장 후반·비율 낮음) |
| **보스: 실험용 실험체** | 느림, 가끔 일직선 돌진, 체력 25% 이하 시 폭주 |

### 2스테이지 — 해저

| 적 | 특징 |
|----|------|
| 피라냐 | 기본 |
| 상어 | 빠른 이동속도, 낮은 체력 |
| 청새치(마를린) | 가장 빠름, 매우 낮은 체력 |
| 전기뱀장어 | 약간 빠름, 체력 높음, 범위 지속 데미지 |
| **보스: 향유고래** | 느림, 체력 많음, 초음파 공격 + 페이즈2 디버프 |

### 3스테이지 — 심해

| 적 | 특징 |
|----|------|
| 심해아귀 | 기본 |
| 대왕문어 | 큰 덩치, 느림(너프 후) |
| 심해해파리 | 자폭 (노란 경고 점멸 + 스케일 펌프) |
| 풍선장어 | 발광 꼬리에서 원거리 공격 |
| **보스: 산갈치** | 매우 긴 몸체, 발광, 이동 방향으로 회전(상하 포함), 돌진/휘두르기 |

### 4스테이지 — 파괴된 연구소 (강화된 전투요원들)

| 적 | 특징 |
|----|------|
| 강화 근접 전투요원 | 빠른 이동속도, 강력한 공격 |
| 강화 방패 전투요원 | 체력 많음, 느림, 와이드 근접 |
| 원거리 전투요원 | 직선 레이저 공격 (원거리 조준) |
| 자폭 드론 | 빠름, 접근 후 자폭 (LED 경고) |
| **보스: 연구소장** | 인간+기계 강화, 드론 소환 + 직선 레이저 + 보호막 |

## 5. 독창적 시스템 (4종)

### 5-1. 압력 시스템
- **적용 스테이지:** 해저(2), 심해(3)
- 하강(Y 좌표)할수록 압력 증가 → 이동속도 및 공격속도 감소 (페널티)
- 대신 경험치 획득량 증가 (리스크/리워드 구조)
- 메타 업그레이드(세포)로 최대 저항 영구 강화 가능
- 답답함 방지: 저항이 낮을 땐 어렵지만 업그레이드하며 극복하는 성장 서사 연출

### 5-2. 공격 수단 카피 시스템
- 스테이지 보스 격파 시 보스의 스킬 3개 중 1개 선택하여 카피
- 각 스킬은 공격형 / 생존형 / 특수형으로 분류
- 카피된 스킬은 슬롯에 장착: **Q(1스) / E(2스) / 스페이스바(3스)**
- 4스테이지 보스(연구소장)는 카피 대상 제외 (마지막 스테이지)

### 5-3. 생체 에너지 시스템
- 적 처치 시 생체 에너지 충전 (최대 200E)
- 카피한 보스 스킬 발동에 소모 (스킬마다 차등: 30E ~ 140E)
- 게이지 형태로 표시 → 언제 쓸지 플레이어가 판단하는 전략적 운용

### 5-4. 돌연변이 진화 시스템
- 특정 레벨(14, 25)에서 총 2회 돌연변이 선택 이벤트 발생
- 매번 **3개의 선택지** 중 1개 선택 / 중복 선택 불가
- 모든 선택지가 하이리스크·하이리턴 구조 (강력한 효과 + 단점 수반)
- 게임 정체성: "불안정한 실험체의 성장" → 안전한 선택지 없음

## 6. 스킬 구성

> 일반 스킬 풀 **12종** (공격 7 + 패시브 5). 레벨업 시 3장 중 1장 선택.
> 슬롯 제한: 공격 5 / 패시브 3. 각 스킬 최대 Lv4.

### 6-1. 공격 스킬 (7)

| 이름 | 타입 | 설명 | 레벨업 특징 |
|------|------|------|------------|
| 휘두르기 | 자동 공격 | 주변 부채꼴(100도) 휘두르기, 기본 스킬 | Lv4: 넉백 |
| 독침 | 원거리 | 가장 가까운 적에게 투사체, 중독 부여 | Lv4: 관통, 투사체 3개 |
| 생체 폭발 | 자동 공격 | 캐릭터 중심 원형 폭발, 스턴 부여 | Lv4: 범위·스턴 증가 |
| 전기 기관 | 자동 공격 | 감전 부여, 인접 3체 연쇄 방전 | Lv4: 사망 시 감전 필드 |
| 가시 산탄 | 자동 공격 | 8방향 가시 동시 발사, 맞은 적 둔화 | Lv4: 출혈 부여 |
| 음파 펄스 | 자동 공격 | 주위 광역 음파 펄스 (긴 쿨다운) | Lv4: 범위·피해 증가 |
| 흡혈 촉수 | 자동 공격 | 가장 가까운 적 1체 라인 데미지 + 흡혈 | Lv4: 흡혈량 증가 |

### 6-2. 패시브 스킬 (5)

| 이름 | 설명 | 레벨업 특징 |
|------|------|------------|
| 세포 재생 | 초당 체력 회복 | Lv4: 초과 회복 → 보호막 |
| 가속 변이 | 이동속도 영구 증가 | Lv4: 적 처치 시 순간 가속 |
| 신경 가속 | 전체 스킬 공격속도 증가 | Lv4: 스킬 추가 발동 확률 |
| 발광 기관 | 주기적으로 1회 피해 흡수(발광 방어막) | Lv4: 흡수 주기 단축 |
| 자기 유도 | 경험치 흡인 범위 증가 | Lv4: 흡인 범위 추가 증가 |

### 6-3. 상태이상 (디버프) — 9주차 구체화 중

- 적에게 부여되는 상태이상을 통합 시스템으로 관리하고, **적 머리 위 아이콘 + 약한 오라**로 시각화
- **중독(독침):** 갱신형 — 재적중 시 지속 갱신 + 즉발 보너스 피해
- **출혈(가시 산탄 Lv4 등):** 누적형 — 스택이 쌓일수록 도트 피해 증가 (최대 스택 캡)
- **둔화 / 스턴:** 갱신형 (가장 강한 값 유지 + 시간 리셋)

## 7. 돌연변이 목록 (5종)

> 레벨 14·25에서 3장 중 1장 선택. 중복 불가. 각 효과는 강력하지만 패널티 동반.

| 이름 | 효과 | 패널티 |
|------|------|--------|
| 과부화 | 공격력 +20% | 최대 HP -30% |
| 과성장 촉수 | 전 스킬 사거리 ×1.6 | 이동속도 -15% |
| 감각 붕괴 | 공격속도 ×1.5, 생체 에너지 충전 ×1.25 | 스킬 사용 시 5% 확률 0.5초 스턴 |
| 의태 기관 | 이동속도 ×2, 40초마다 3초 무적 | 공격력 -50% |
| 독성 과부화 | 독/감전 피해 ×1.7 | 물리 피해 ×0.7, 물리 범위 ×0.8 |

> ※ 독성 과부화의 "상태이상 추가 피해"는 9주차 디버프 시스템 도입과 함께 적용 예정.

## 9. 메타 업그레이드 (세포)

- **재화명:** 세포
- **획득:** 게임 진행 중 만렙 보상 / 보스 처치 / 스테이지 클리어로 누적, 패배·완료 후 성적 반영
- **사용처:**
  - 기본 스탯 영구 강화 (최대 HP, 이동속도, 공격속도, 공격력)
  - 압력 저항 강화 (단계별 업그레이드)
- 추후 시간 여유에 따라 스킬 해금 시스템 추가 고려

## 10. 캐릭터 비주얼

- **종류:** 심해 생체 실험체 (촉수형 괴물)
- **핵심 특징:** 두족류(문어/오징어) 기반
  - 여러 갈래의 촉수, 생체 발광 외눈
  - 잠수정 잔해를 두른 듯한 실루엣 (AI 생성 단일 sprite)
  - 별도 프레임 시트 없이 **절차적 애니메이션**으로 생동감 부여
- **색상:** 짙은 청록 + 흰/청록 발광 포인트
- **피부:** 반투명, 생체 발광 점 패턴
- **절차적 모션:** 부유하는 보빙(상하) + 호흡 스쿼시 + 입력 방향 기울기 / 좌우 플립 (넉백에도 바라보는 방향 유지)

### 돌연변이별 비주얼 (컨셉)

> 현재는 스탯 변화 중심. 의태 기관 무적 시 반투명 시각화만 구현됨. 아래는 추후 확장 컨셉.

| 돌연변이 | 비주얼 컨셉 |
|---------|------------|
| 과부화 | 몸이 붉게 발광 |
| 과성장 촉수 | 촉수가 여러 갈래로 길게 분기 |
| 감각 붕괴 | 몸 전체 균열 + 파란 발광 |
| 의태 기관 | 몸이 반투명해짐 (무적 시 구현됨) |
| 독성 과부화 | 촉수에서 독액이 흘러내림 |

### 주요 기술 구현 방향

- **자동 공격:** Physics2D OverlapCircle로 범위 내 적 탐색, 가장 가까운 적 자동 타겟팅
- **적 AI:** 벡터 추적 방식, ScriptableObject로 적 데이터 관리, 풀링(Object Pool) 스폰
- **스킬 시스템:** 각 스킬을 독립 컴포넌트로 모듈화, 레벨별 수치 ScriptableObject 분리
- **압력 시스템:** 플레이어 Y 좌표 기준 압력 계산, 이동/공격속도에 곱연산
- **심해 시야 제한(3스):** URP 색보정 + 강한 비네트 + 자동추적 사거리 상한
- **동적 스폰:** DifficultyManager가 시간 기반 선형 램프 + 보스 직전 위기 배율 / 보스 등장 완화 배율 적용
